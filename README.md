# ساخت آواتار واقعی با Google Flow

راهنمای فارسی صفر تا صد ساخت ویدیوی آواتار با عکس‌های واقعی، Gemini Omni Flash، دیالوگ فارسی، تنظیم جهت نگاه، Voice، Character و رفع مشکلات حفظ چهره.

## نسخه آنلاین

https://soheil78ai.github.io/characters/

## محتوا

- انتخاب عکس‌های مرجع مناسب
- تنظیم دقیق Video / Ingredients یا Components / 9:16 / Omni Flash
- ساخت صدای فارسی و Voice Performance
- پرامپت کامل همان سبک ریلز «من و نسخه هوشمندم»
- قالب عمومی قابل شخصی‌سازی
- ساخت Character ثابت
- استفاده از Avatar رسمی Flow با `@me` در مناطق پشتیبانی‌شده
- رفع مشکل تغییر چهره، حذف دیالوگ و جهت نگاه اشتباه
